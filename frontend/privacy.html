Privacy policy

Last updated: 2026-05-25

Français

En bref

Helios-Lidar respecte ta vie privée. Aucun cookie, aucun script de tracking, aucune empreinte numérique n'est posé sur ton navigateur. Le code source du site et de la carte est entièrement accessible sur GitHub : ce qui est écrit ici se vérifie ligne par ligne dans le dépôt.

Ce qu'on garde

  • Logs serveur nginx : adresse IP, user-agent, page demandée, horodatage, page de provenance. Ce sont les logs techniques standard d'un serveur web, indispensables pour la sécurité (anti-abus) et le debug. Rétention : 30 jours maximum, ensuite ils sont effacés automatiquement.
  • Compteur de conversions : un simple compteur entier est incrémenté à chaque conversion réussie. Aucune information sur le contenu, l'utilisateur ou la zone géographique convertie n'est associée à ce compteur.
  • Cache pays : pour afficher la carte des pays visiteurs sur le dashboard /stats, l'IP est envoyée à ip-api.com qui renvoie un code pays. Seul le code pays est conservé (pas l'IP ni la position précise).

Ce qu'on NE garde PAS

  • Tes fichiers convertis ne sont jamais sauvegardés. Le COG généré (.tif) reste sur le serveur uniquement le temps que ton navigateur le télécharge, avec un timeout de 10 minutes après quoi le fichier est effacé automatiquement, qu'il ait été téléchargé ou non. Aucune statistique n'est extraite du contenu de ton fichier (à part le simple +1 sur le compteur de conversions).
  • Aucun cookie n'est posé.
  • Aucun script tiers de tracking (pas de Google Analytics, pas de Facebook Pixel, etc.).
  • Aucun fingerprinting (canvas, WebGL, fonts, etc.).
  • Aucune revente, aucun partage de tes données à un tiers à des fins marketing.

Tes droits

Tu peux à tout moment demander la suppression des entrées de logs liées à ton IP en m'envoyant un email à jcremoux@gmail.com. Comme les logs s'effacent automatiquement après 30 jours, attendre suffit aussi.

Base légale

Les logs serveur et le geo-lookup sont conservés au titre de l'intérêt légitime (art. 6.1.f du RGPD) : sécurité du service et statistiques d'usage agrégées non-individuelles. Aucun consentement explicite n'est requis pour ces opérations.

Hébergement

Le serveur est loué chez OVH (France), un hébergeur soumis au droit français et européen.

English

TL;DR

Helios-Lidar respects your privacy. No cookie, no tracking script, no fingerprint is set on your browser. The full source code of the site + the card is open on GitHub: everything written here can be verified line by line in the repository.

What we keep

  • Nginx server logs: IP address, user-agent, requested path, timestamp, referrer. Standard technical logs every web server keeps, needed for security (abuse mitigation) and debugging. Retention: 30 days max, after which they're automatically purged.
  • Conversion counter: a simple integer counter is incremented after each successful pipeline run. No information about the content, the user or the converted region is associated with that counter.
  • Country cache: to render the visitors-by-country chart on the /stats dashboard, the visitor IP is sent to ip-api.com which returns a country code. Only the country code is stored (not the IP, not the precise location).

What we do NOT keep

  • Your converted files are never saved. The generated COG (.tif) stays on the server only long enough for your browser to download it, with a 10-minute timeout after which the file is automatically deleted, whether downloaded or not. No statistic is extracted from the content of your file (beyond the simple +1 on the conversion counter).
  • No cookie is set.
  • No third-party tracking script (no Google Analytics, no Facebook Pixel, no anything).
  • No fingerprinting (canvas, WebGL, fonts, etc.).
  • No reselling, no sharing of your data with any third party for marketing.

Your rights

You can ask for the log entries linked to your IP to be removed at any time by emailing jcremoux@gmail.com. Since logs auto-purge after 30 days, waiting is equally fine.

Legal basis

Server logs and geo-lookup are retained under legitimate interest (GDPR Art. 6.1.f): service security and aggregated, non-individual usage statistics. No explicit consent is required for these operations.

Hosting

The server is rented from OVH (France), a host subject to French and European law.
